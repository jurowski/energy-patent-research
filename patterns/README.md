## The Meta-Pattern

The most promising devices combine three elements:
1. **Non-linear element** (saturating core, plasma gap, ferroelectric)
2. **Resonance** (LC circuit, mechanical, acoustic)
3. **Pulsed excitation** (sharp edges, capacitor discharge, impulse)

See [10_synthesis.md](10_synthesis.md) for the full synthesis and a
prioritized experiment hierarchy.

## Total Experiments Defined: 25+

Each pattern file contains 2-4 experiments with complete materials lists,
procedures, and success criteria. Start with Level 1 (single-element)
experiments and work up to Level 3 (three-element combinations).

Estimated cost for Level 1-3 experiments: **$200-500 total**.
