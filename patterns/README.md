# Pattern Analysis

This directory catalogs recurring technical patterns found across
high-efficiency energy device patents.

Each pattern file should document:
- What the pattern is
- Which patents/devices exhibit it
- The theoretical basis (mainstream or alternative)
- How to test it experimentally
- Bill of materials for a simple test rig

## Patterns identified so far

(To be populated as research progresses)

- `resonance.md` — Electrical/mechanical resonance exploitation
- `pulsed_dc.md` — Pulsed DC waveforms vs sinusoidal AC
- `bifilar_coils.md` — Tesla's bifilar coil and derivatives
- `magnetic_flux_switching.md` — Rapid flux path changes
- `radiant_energy.md` — Cold electricity / non-Hertzian claims
- `lenr_patterns.md` — Common features of LENR/cold fusion devices
